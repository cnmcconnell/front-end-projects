jQuery Glossary

val() - retrieves the value of a text box
text() - sets the value of a text box
next() - gets the next sibling of an element
submit() - submits the form
focus() - moves the focus to a selected control/link
prev() - gets the previous sibling of selected element
find() - searches the element and returns descendent elements
attr() - retrieves value of specific attribute
css() - sets specific property value
addClass() - adds class to element
removeClass() - removes class from element
toggleClass() - removes class if present and vice versa
html() - sets HTML content of element
hide() - hides elements
show() - shows elements
each() - runs each element's function in array
ready() - runs when the DOM is ready
click() - runs when an element is clicked on
mouseover() - runs when mouse is over an element
hover() - runs when mouse is over an element and stops when mouse leaves
animate() - changes the properties of a selected element to whatever the new specified properties are in a specified amount of time
delay() - this will delay the next animation from starting
stop() - this causes the current animation to stop
finish() - causes all queued animations to set to their end values
